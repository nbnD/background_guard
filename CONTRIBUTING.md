## How to Contribute

1. Fork the repository
2. Create a new branch from `main`
3. Make your changes
4. Ensure the code builds and tests pass
5. Open a Pull Request (PR) to the `main` branch

Direct pushes to `main` are not allowed.

---

## Contribution Guidelines

- Keep changes focused and minimal
- Follow existing code style and structure
- Do not introduce breaking changes without discussion
- Add documentation or comments where helpful
- Avoid adding unnecessary dependencies

---

## Pull Request Review Process

All pull requests are reviewed by the maintainers.

A PR may be accepted if it:
- Solves a real problem
- Is well-structured and readable
- Aligns with the project goals

The maintainers may request changes or close PRs that do not align with the project direction.

---

## Ownership and Control

This project is open source under the MIT License.

While contributions are welcome:
- Final merge decisions remain with the maintainers
- Submitting a PR does not grant push or admin access
- Maintainers reserve the right to accept or reject contributions
